# Deploy Steps - Simple Hinglish Guide

## 1. Zip extract karo
`goyal-attendance-app.zip` ko extract karo.

## 2. GitHub me upload
GitHub par New Repository banao, naam:

`goyal-attendance-app`

Extracted folder ke files upload karo.

## 3. Vercel me deploy
- Vercel open karo
- Add New Project
- GitHub repo select karo
- Framework: Other / Static
- Deploy dabao

## 4. Mobile me test
Vercel link mobile Chrome me kholo:

- 3 dots
- Add to Home Screen

## 5. Admin login

## 6. Pehla kaam admin panel me
- Locations me Office/Godown add karo
- Employees me employee add karo
- Employee type choose karo:
  - Office: time restriction ON
  - Godown: time restriction OFF
  - Field: geofence OFF
